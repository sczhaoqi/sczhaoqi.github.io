vscode与eslint

 2021-06-29 |  编程 · eslint · vscode · 前端 ·  |  program / 前端 / 

1.安装插件

eslint vetur

2. 开启eslint-format

扩展-> eslint -> 扩展设置 Eslint>Format:Enable

或者 直接编辑setting.json，添加

    “eslint.format.enable”: true

3. 打开.vue文件

alt+shift+f 快速格式化 选择eslint作为格式化工具
