element-ui默认样式修改

 2019-03-07 |  vue · element-ui ·  |  program / 前端 / 

我们在使用element-ui的时候经常会遇到需要修改组件默认样式。

我目前知道的有两种方法：

1.通过创建一个css文件，然后在某一个组件中引用，css文件中使用和组件样式一样的命名，这样组件的默认样式就会被覆盖。

2.使用  /deep/

.homePage /deep/ .el-main {

padding: 0;

}

  .homePage为我们要修改组件类名的父级组件样式类名。。即使定义一个空的类名也没事

  .el-main为我们要修改组件的样式。
